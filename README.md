# 📊 ECサイトにおける消費者行動分析と商品推薦

---

## ■ 概要

本プロジェクトでは、ECサイトのユーザー行動データを用いて、

- 消費者がどのような商品に興味を持つのか
- どのような条件で購買に至るのか
- どのような商品が売れやすいのか

を分析する。

最終的には、分析結果をもとに商品推薦やECサイト改善への提案を行うことを目的とする。

---

## ■ 分析目的

ECサイト上におけるユーザー行動を分析し、消費者の興味・購買傾向を把握する。

特に以下の行動から、購入に至るプロセスを整理する：

- view（閲覧）
- wishlist（ウィッシュリスト追加）
- cart（カート追加）
- purchase（購入）
- review（レビュー）

これらをもとに、ユーザーの意思決定構造を明らかにする。

---

## ■ 使用データ

Kaggle：E-commerce_dataset

| データ | 内容 |
|------|------|
| users | ユーザー情報 |
| products | 商品情報 |
| orders | 注文情報 |
| order_items | 購入商品情報 |
| reviews | レビュー情報 |
| events | ユーザー行動ログ |

## ■ 分析方針

### ■ 想定特徴量

- 購買数
- レビュー評価
- 価格
- view数
- wishlist数
- cart数

---

### ■ 初期仮説

- 「購買数とレビュー評価が高い商品は人気商品である」

---

## ■ 分析ロードマップ

### Phase1：人気商品の把握

- 購買数上位商品
- 高評価商品
- 人気商品の特徴抽出

---

### Phase2：消費者行動分析

| 行動 | 意味 |
|------|------|
| view | 興味 |

| cart | 購買意欲 |

---

### Phase3：売れやすい商品の条件

- 価格帯

- レビュー傾向
- 行動パターン

### Phase4：推薦・改善提案

- 商品戦略提案
